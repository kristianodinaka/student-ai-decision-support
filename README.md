# Student AI Decision Support System

## Overview
The Student AI Decision Support System is a web application that helps students make informed decisions by analyzing multiple options and providing AI-powered recommendations.

## Features
- AI-powered decision analysis
- User-friendly interface
- Real-time responses
- Responsive design
- Accessible on desktop and mobile

## Technologies Used
### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- Node.js
- Express

### Deployment
- Frontend: Vercel
- Backend: Render

## Installation

Clone the repository:

```bash
git clone https://github.com/kristianodinaka/student-ai-decision-support.git
```

Install dependencies:

```bash
cd frontend
npm install

cd ../backend
npm install
```

Run locally:

Frontend:

```bash
npm run dev
```

Backend:

```bash
node server.js
```
